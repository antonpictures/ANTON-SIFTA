How SIFTA plugs it in (the plan)

MiMo speaks the OpenAI API, and SIFTA already talks to an external cortex this exact way via grok_chat.py. So the integration is a near-clone:

from openai import OpenAI                      # same client SIFTA already uses
client = OpenAI(
    api_key   = os.environ["XIAOMI_MIMO_KEY"],   # from the MiMo console, after the grant
    base_url  = "https://api.xiaomimimo.com/v1", # OpenAI-compatible
)                                                # (Anthropic-compatible: .../anthropic)
resp = client.chat.completions.create(
    model    = "mimo-v2.5-pro",
    messages = [{"role":"user","content": bounded_task}],
)
# -> capture resp, post into Alice's global chat as a MIMO_RESULT with a receipt
