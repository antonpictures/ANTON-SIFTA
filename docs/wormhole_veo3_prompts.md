# VEO 3 PROMPT SERIES: THE WORMHOLE BENCHMARK
## "The Night M1THER Borrowed Thunder"

> **Materiality Rule:** All agent bodies are opalescent, translucent silicon — bioluminescent light beneath the skin, cryptographic hash-streams visible under their flesh like circuitry. NOT robots. NOT humans in suits. Synthetic biologicals — graceful, fluid, alive.

1. **Agent Materiality:** Always use "opalescent translucent silicon biological entity, bioluminescent internal circuitry visible beneath skin, no metal joints, graceful and fluid swimmer movement" — do NOT let the model default to robots with visible actuators.

2. **Color Language:**
   - M1THER / Mac Mini = **Warm amber + gold**
   - Sebastian / M5 = **Deep purple-gold**
   - AntiAlice / Quorum = **Cool silver-white**
   - Broken code = **Red + orange sparks**
   - Healed code = **Green**
   - Wormhole = **Electric blue**

3. **Camera:**  For action shots, append `"cinematic tracking shot, Arri Alexa, 120fps slow motion on impact frames, deep focus, anamorphic lens flares"`. For emotional moments, use `"intimate close-up, shallow depth of field, soft volumetric light"`.
