# 🐜⚡ SIFTA PDF Forge

A small hand in Alice's body: a deterministic render organ (§7.2) that turns dropped
assets + copy into a finished one/two-page PDF flyer — the same layout I built for the
WIN-WIN flyer. No internet, no install. The cortex still writes the words when you ask
it; this organ just forges the page.

## Use it
1. Double-click **`Open PDF Forge.command`** (or open `PDF_Forge.html` in Safari/Chrome).
2. **Drop images** on the left. First image = the hero (page 1). The next images become
   the renderings gallery (page 2). Page 2 only appears when you add 2+ images.
3. **Edit the copy** in the fields, or paste a block of text into **Prompt / paste content**
   and press **Auto-fill** (4 blank-line-separated paragraphs map to the 4 cards).
4. Press **Forge PDF →**. In the print dialog choose **Save as PDF** → save.

## Notes
- The page renders at US Letter, margins 0 — vector-crisp text, exactly like the WIN-WIN PDF.
- **Reset copy** restores the WIN-WIN defaults so you always have a working example.
- Everything stays local in the browser; nothing is uploaded.
